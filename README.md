# Hixme
Persons Data API

## Usage

1. Run Locally

```
npm install
```
```
npm start
```
```
curl -X GET http://localhost:8080/api/summary
```

2. Google App Engine
```
curl -X GET https://hixme-234319.appspot.com/api/summary
```

## Testing

* Run tests
```
npm test
```

* Code coverage
```
npm run nyc
```
